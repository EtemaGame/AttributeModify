# AttributeModify

Modify item attributes through datapacks. Supports mining speed, mining tier, durability, and optional Curios or Accessories integration.

## Features

- Add, modify, or remove item attributes
- Override mining speed and mining tier
- Override durability and custom durability triggers
- Works with vanilla and modded items
- Multiplayer friendly and `/reload` compatible

## Datapack Location

Place files in:

```text
data/<namespace>/item_attributes/<file>.json
```
